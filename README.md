## Installing

A step by step series of examples that tell you how to start the app

Download the proyect

```
git clone https://github.com/Vente16/triki-game/
```

Get in the project

```
cd triki-game
```
Install yarn

```
npm install yarn 
```

Run the app

```
npm start 
```

Or
```
yarn start 
```
